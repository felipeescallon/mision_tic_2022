En 15 minutos:

1. Entrar a la página y descargar XAAMP
2. Instalar XAAMP (siguiente, siguiente, ....)
3. Abrir XAAMP Control Panel
4. Click en Start (Apache y de MySQL)
5. Abrir una página web y digitar: localhost
6. Click en phpMyadmin
7. Crear base de datos poniendo el nombre de: mascotas
